# Support

## Beta Release

**SPFx Local Workbench** is currently in **beta** and under active development. Features are being added and refined regularly, and breaking changes may occur between releases.

## No Formal Support

At this time, **no formal support is provided** for this extension.

## Reporting Issues

While we cannot guarantee support, we welcome feedback and issue reports:

- **Check existing documentation** - Review the [README](README.md), [PROXY.md](PROXY.md), [MOCK-DATA.md](MOCK-DATA.md), and [TROUBLESHOOTING.md](TROUBLESHOOTING.md) for guidance
- **Verify your setup** - Ensure you're using SPFx 1.22+ with Heft (not Gulp)
- **Community feedback** - Issues and feedback help improve the extension for everyone

**Note**: The extension is provided as-is for development and testing purposes.
